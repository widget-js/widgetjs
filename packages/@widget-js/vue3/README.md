## 桌面组件 Vue

### 项目目录结构

```
widgets
├── src
│   ├── components                // 常用Vue组件
│   └── index.ts
├── .gitignore
├── package.json
├── README.md
└── tsconfig.json
```

#### 依赖下载

```
// 下载依赖可能需要翻墙
yarn install
```

#### 单独启动Vue

```
yarn serve
```

#### 项目主要依赖

[Element Plus](https://element-plus.gitee.io/zh-CN/component/button.html)
[MingCute图标库](https://www.mingcute.com/)
